## 시간복잡도

### 1초 기준 최대 연산 횟수
- 일반적인 CPU 기준 **최대 연산횟수는 약 1억(10⁸) 연산**

### 시간 복잡도별 허용 범위

| 시간 복잡도 | 1초 내 가능한 N |
|------------|----------------|
| O(1) | 무관(1억번) |
| O(log N) | 매우 큼 |
| O(N) | 100,000,000 |
| O(N log N) | 1,000,000 |
| O(N²) | 10,000 |
| O(N³) | 500 |
| O(2ᴺ) | 20 |
| O(N!) | 10 |

### 입력 크기(N)에 따른 선택 기준

- **N ≤ 500** → `O(N³)` 이하
- **N ≤ 2,000** → `O(N²)` 이하
- **N ≤ 100,000** → `O(N log N)` 이하
- **N ≤ 10,000,000** → `O(N)` 이하

### 시간 복잡도 크기 비교
O(1) < O(log N) < O(N) < O(N log N) < O(N²) < O(2ᴺ) < O(N!)

> **입력 크기(N)를 먼저 보고  
> 시간 복잡도를 결정한 뒤  
> 알고리즘을 고른다.**
